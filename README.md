# Phishing Email Detection Model

## Overview

This project uses Machine Learning and Scikit-learn to classify emails as either:

* Phishing
* Safe

## Features

* Trains on phishing and legitimate email samples
* Extracts textual features using CountVectorizer
* Uses Naive Bayes classification
* Displays prediction accuracy
* Displays confusion matrix
* Predicts whether a new email is phishing or safe

## Technologies Used

* Python
* Scikit-learn
* Pandas

## How to Run

python phishing_detector.py

## Expected Outcome

The model classifies emails as Phishing or Safe and evaluates performance using accuracy and confusion matrix.
